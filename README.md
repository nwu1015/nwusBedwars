# BedWars Plugin

Цей проєкт є базовою реалізацією плагіна **BedWars** для серверів **Minecraft Spigot (1.21+)**.  
Плагін має модульну структуру для зручного розширення (команди, слухачі, менеджери, конфігурації тощо).

---
## Основні модулі

| Модуль | Призначення |
|--------|--------------|
| `BedWarsPlugin` | Точка входу в плагін, реєструє команди та слухачі. |
| `commands` | Реалізація команд `/start`, `/join`. |
| `game` | Логіка гри: арени, команди, стани, керування ліжками. |
| `listeners` | Реакція на події гравців (вхід, вихід, знищення блоку). |
| `utils` | Допоміжні класи, наприклад, для кольорових повідомлень у чаті. |
| `config` | Робота з файлом `config.yml` і зчитування налаштувань. |

---

## Встановлення та запуск

### Попередні вимоги
- Java **17** або новіше  
- Сервер **Spigot** версії **1.21+**  
- **Maven** (для компіляції)

---

У проєкті виконай: Build -> Build Project.
Після збірки з’явиться файл: nwusBedwars-1.0.0.jar
